AI Smart Attendance System

AI-powered attendance management platform developed for MIT Vishwaprayag University, Solapur. The system automates attendance tracking using facial recognition and provides complete academic attendance management.

Key Features

  • ✅ Face Recognition Attendance
  • ✅ Student Registration & Face Enrollment
  • ✅ Attendance Dashboard
  • ✅ Timetable Management
  • ✅ Subject Management
  • ✅ Attendance Leaderboard
  • ✅ Parent Email Notifications
  • ✅ Signup Request Approval System
  • ✅ Excel Report Export
  • ✅ Batch & Course Management

Technology Stack

HTML5 CSS3 JavaScript Node.js Express.js MongoDB Face Recognition AI ExcelJS
